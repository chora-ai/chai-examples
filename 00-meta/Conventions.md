# Conventions

Conventions are rules about form — distinct from process workflows (which live in [[Workflows]]).

## Quick Conventions

- Notes use title case (e.g., `AI Assistant.md`, `Private Git Host.md`)
- Meta subdirectory files use lowercase (e.g., `general.md`, `daily.md`)
- Within-folder wikilinks use bare links: `[[Note Name]]`
- Cross-folder wikilinks include path prefix: `[[subfolder/Note Name|Note Name]]`
- No wikilinks to inbox notes — only reference by inbox notes by name
- When referencing a note in prose, include its type (e.g., "the AI Assistant project note")
- Frontmatter `tags` are for topics/domains only — never duplicate `type` or `status`
