# Daily Note Conventions

Conventions for daily notes in `00-daily/`. For the daily workflow process, see [[Workflows]].

## Section Order

Daily notes follow this section order:

1. Action Items
2. Triage
3. Daily Insights
4. Looking Ahead (Top Priorities / On the Horizon)

Daily notes are reflective and forward-looking — session logs do not belong in daily notes.

## Triage Entries

Triage entries are limited to notes that have a `status` frontmatter field. Each entry should not exceed 200 characters. Format: `- Reviewed: [[path/Note|Note]] → status kept/changed, brief reason`.

## Daily Insights

Insights should be standalone sections with descriptive headings — no "Additional Insights" catch-all. Ask "what does this mean?" not "what happened?"
