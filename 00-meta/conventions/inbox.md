# Inbox Note Conventions

Conventions for inbox notes in `00-inbox/`.

## Purpose

Inbox notes are scratchpads for quick capture and refinement. No frontmatter — they're meant to be processed and either absorbed into existing notes or distributed into new ones, then deleted. Structure comes when the idea finds a home, not before.

## Referencing Inbox Notes

- No wikilinks to any note inside `00-inbox/` — inbox notes are transient and links break when the note is processed and deleted
- Reference inbox notes by name with a type label in prose: e.g., "the Ideas inbox note"
- Only reference inbox notes from daily notes — project notes should not mention inbox notes
