# Daily Note Workflow

Daily notes are reflective and forward-looking documents. They capture meaning and direction, not operational logs. For format rules (section order, triage entries, insights), see [[conventions/daily|daily conventions]].

## Flow

1. **Start of day** — create a new note in `00-daily/` using the `daily` template; add action items for the day
2. **Throughout the day** — check off completed items; triage 3-5 notes; add insights as new context emerges
3. **End of day** — add Looking Ahead section (Top Priorities and On the Horizon)

## Sections

| Section | Purpose | When to add |
|---|---|---|
| Action Items | Checklist for the day | Start of day; update throughout |
| Triage | Status decision record for notes with `status` frontmatter | Throughout the day |
| Daily Insights | Reflective observations and analysis | Throughout the day; grows incrementally |
| Looking Ahead | Top Priorities and On the Horizon | End of day |
