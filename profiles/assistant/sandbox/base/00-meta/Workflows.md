# Workflows

Workflows are processes to be followed — distinct from formatting conventions (which live in [[Conventions]]).

## Quick Reminders

- Daily notes: create at start of day, triage throughout, Looking Ahead at end of day
- Triage 3-5 notes per session — prioritize those actively referenced or with stale sentiments
- When new insights emerge, add them to the daily note
