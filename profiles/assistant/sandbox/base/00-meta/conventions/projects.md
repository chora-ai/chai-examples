# Project Note Conventions

Conventions for project notes in `01-projects/`.

## Section Order

Project notes follow this section order:

1. Sentiment — current feeling or posture toward the project
2. Action Items — checklist of concrete tasks
3. Sub-Notes — wikilinks to child documents in the note's subfolder
4. External Links — URLs to project resources

A horizontal rule separates the header (above sections) from body content (note-specific prose). Not all project notes have body content — the separator appears only when body sections exist.

## Sub-Notes

Project notes list their child documents in the Sub-Notes section. Sub-notes live in a same-named subfolder (e.g., `Sample Project/Design.md`). Sub-notes omit `type` — the filesystem location expresses the parent relationship. Use `- NA` when there are no sub-notes.
